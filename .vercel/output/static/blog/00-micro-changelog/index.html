Pagefind search üîé

Pagefind is a search library for static websites. Micro uses Sergey Shishkin‚Äôs astro-pagefind integration. This integration simplifies development and does not require any changes to the default build script.

Press / or CTRL + K to open the search dialog. For Mac users, ‚åò + K can also be used. To dismiss the search dialog, press Esc or click on an area outside the component.

Build and develop

üö®

The site must be built at least once so Pagefind can index the content.

# Pagefind must index the site to function
npm run build

When developing you can continue to use npm run dev and Pagefind will use the index from the last available build.
